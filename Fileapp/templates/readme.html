PepTag_MSappの説明

ペプチドタグのMS解析結果を集計してグラフやExcelファイルとして返します。
利用するには研究室の資料を参照して、測定結果のファイル(.xlsx)ごとにPre_Qシートを作成してください。

①ファイルのアップロード

「ファイルを選択」をクリックして、集計するファイル(.xlsx)を選択します。
「アップロード」をクリックするとファイルが集計対象のリストに入ります。
選択するファイルを誤った場合は個別または一括でリストから削除できるので、
同様の手順で登録しなおしてください。

②グラフの作成

①で集計対象としたファイルをPrecursor ionの種類ごとに平均します。
全てのionを測定回数で平均するため１回しか測定できていないionも含まれます。
Pre_Qシート以外は読み込むことが出来ないため集計することができません。
（解析したいExcelのシートをPre_Qという名前にすれば解析できるかも。）
ペプチドタグはChargeが"2"のもののみが描画されます。

1. グラフのタイトル：
グラフのタイトルを指定できます。英数字のみ対応。
18文字を越えたあたりからおさまりが悪くなります。
タイトルなしのグラフも作成可能です。

2. Proteome Discovererのバージョン：
バージョン毎に検出強度の呼び方が異なるため使用したバージョンを選択してください。
解析に使用したバージョンと異なるバージョンを選択するとエラーになります。

3. 縦軸のオーダー：
ペプチドタグの精製試料と精製前の試料ではインジェクションされる量が異なるため、
検出強度を調整することでインジェクション量を実質的に同量にします。
精製試料では10000、精製前の試料では1000にすることを推奨します。

4. ケラチンの除去：
集計からケラチンを除きたかったらyesを押してください。
どっちでもいいと思います。

「グラフの作成」をクリックすると集計および散布図の作成を行います。

③Light/Heavyの集計

PD13を使用した際に利用できます。
ペプチドタグのデータを抽出します。その後の統計解析はExcelで行ってください。
ペプチドタグの検出強度の最大値の抽出では、Modificationを無視して検出強度が最も高いものを抽出します。
